Propiedades y teoremas:
  - En promedio los numeros tienen n ^ (1 / 3) divisores.
  - Sea d(i) el numero de divisores de i, la sumatoria de d(i) para i entre 1 y n
    es aproximadamente n*ln(n)
  - Entre 1 y n hay aproximadamente pi(n) = n / ln(n) numeros primos
  - La distancia promedio entre primos desde 1 hasta n es es ln(n).
    Para todos los enteros hasta 10⁹, la maxima distancia entre primos es 282. 
    Hasta 10¹⁸ es 1442
  - Teorema de Bertrand dice que para todo n >= 2 hay al menos un primo n < p < 2n

  Descomposicion prima de un numero n = p_1 ^ a_1 * p_2 ^ a_2 * ... * p_k ^ a_k
  - El numero de divisores de un numero n es:
    t(n) = productorio(ai + 1)
  - La suma de factores de n es:
    o(n) = productorio( ((pi ^ (ai + 1)) - 1) / (pi - 1) )
  - El producto de los factores de n es:
    u(n) = n ^ (t(n) / 2)
  - Un numero es llamado perfecto si n = o(n) - n, es decir,
    si la suma de sus factores entre 1 y n-1 es igual a si mismo

  - Una funcion multiplicativa es aquella en la cual si a, b son coprimos entonces
    f(a*b) = f(a)*f(b)
  - Las funciones d(n) y o(n) son multiplicativas.

  Euler's totient function: phi(n) da la cantidad de numeros coprimos con
    n entre 1 y n. Ej: phi(12) = 4, ya que 1, 5, 7 y 11 son coprimos con 12.
    phi(n) = productorio( (pi ^ (ai - 1)) * (pi - 1) )
  - Si p es primo, phi(p) = p-1
  - Si p es primo y k >= 1, phi(p ^ k) = (p ^ k) - (p ^ (k-1))
  - Si a y b son coprimos, phi(a*b) = phi(a) * phi(b)
  - Para cualquier par {a, b} con d = gcd(a, b), phi(a*b) = phi(a) * phi(b) * d / phi(d)
  - Para cualquier numero n, la sumatoria phi(d) siendo d los divisores de n, es igual a n.
  - * Para calcular (a ^ (b ^ c)) mod m se hace:
      m1 = phi(m), (a ^ ((b ^ c) mod m1)) mod m.
    *  Asi mismo, para calcular (a ^ (b ^ (c ^ d))) mod m se hace:
      m1 = phi(m), m2 = phi(m1), (a ^ ((b ^ ((c ^ d) mod m2)) mod m1)) mod m.
      Y asi sucesivamente para cadenas de potencias.
    * La generalizacion es: (a ^ n) % m = (a ^ (n % phi(m))) % m

  - El pequeño teorema de Fermat dice que (x ^ (m-1)) % m = 1. 
    Siendo m primo y x, m coprimos.
    Además, (x ^ k) % m = (x ^ (k % (m-1))) % m
  - Y de manera mas general el teorema de Euler dice que (x ^ (phi(m))) % m = 1
    cuando x, m son coprimos.

  - Si el inverso modular existe se puede calcular usando a⁻¹ = a^(phi(m)-1)
    y si m es primo entonces x⁻¹ = x^(m-2)

  - El teorema de los numeros poligonales de fermat dice que todo numero puede 
    representarse como la suma de 3 numeros triangulares, 4 numeros cuadrados, 5
    numeros pentagonales y n numeros n-poligonales.
  - Las ternas pitagoricas son ternas (a, b, c) que satisfacen el teorema de pitagoras
    a² + b² = c², lo cual significa que hay un triangulo rectangulo con lados de
    longitud a, b y c, ej: 3, 4, 5.
    Si a, b, c son una terna pitagorica, todas las ternas de la forma (ka, kb, kc),
    con k > 1, tambien lo son.
    Una terna pitagorica es primitiva si a, b y c son coprimos (gcd(a, b, c) = 1),
    todas las ternas pitagoricas pueden ser generadas a partir de estas usando un
    factor k.
    La formula de Euclides se puede utilizar para producir todas las ternas pitagoricas
    primitivas de la forma (n² - m², 2nm, n² + m²), donde 0 < m < n, n y m son copirmos 
    y al menos uno de los dos es par.
    La terna pitagorica primitiva mas pequeña es (3, 4, 5).
  - El teorema de Wilson dice que un numero es primo si y solo si (n - 1)! % n = n - 1

  Fibonacci:
  - Empieza con F_0 = 0, F_1 = 1
  - F_n = ((((1 + sqrt(5)) / 2) ^ n) - (((1 - sqrt(5)) / 2) ^ n)) / sqrt(5)
    Sin embargo es dificil usarla por el redondeo.
  - Con matrices se puede calcular asi:
    (1 1)^n = (F_n+1  F_n )
    (1 0)     ( F_n  F_n-1)
  - Identidad de Cassini: F_n-1 * F_n-2 - F_n² = (-1)^n
  - Regla de adicion: F_n+k = F_k * F_n+1 + F_k-1 * F_n
    Y cuando k = n: F_2n = F_n * (F_n+1 + F_n-1)
    Por lo tanto, para cualquier entero positivo k, F_kn es multiplo de F_n y si
    F_m es multiplo de F_n, entonces m es multiplo de n
  - Identidad del GCD: gcd(F_m, F_n) = F_gcd(m, n)
  - Para cualquier numero p, existe una secuencia de Fibonacci modulo p que se repite 
    periodicamente, de longitud maximo p²
  - Teorema de Zeckendorf: Todo numero entero positivo tiene una unica representacion 
    como una suma de numeros de Fibonacci tal que ningun par de numeros es igual ni
    son 2 numeros de Fibonacci consecutivos. A partir de esto se calcula el codigo de
    Fibonacci.
