# Notebook CodeDeGrace 📓

Notebook de algoritmos y estructuras de datos del equipo **CodeDeGrace**, de la Universidad Tecnológica de Pereira (UTP), usado principalmente en **ICPC** (International Collegiate Programming Contest).

- **Sergio Moncada**

## Contenido

Este repositorio recopila implementaciones probadas y listas para usar durante competencias:

| Carpeta | Contenido |
|---|---|
| `Avanzados` | Algoritmos y técnicas avanzadas |
| `DP` | Programación dinámica |
| `Geometria` | Algoritmos de geometría computacional |
| `Graphs` | Algoritmos sobre grafos |
| `Java` | Implementaciones y utilidades en Java |
| `Math` | Matemáticas y teoría de números |
| `Mazes` | Algoritmos sobre laberintos y grillas |
| `Miscelanea` | Utilidades y snippets varios |
| `Orden y busqueda en vectores` | Algoritmos de ordenamiento y búsqueda |
| `Range Queries` | Estructuras para consultas de rango (segment trees, BIT, etc.) |
| `Strings` | Algoritmos sobre cadenas |

También incluye `template.h`, la plantilla base de C++.

## Uso

Las implementaciones están en C++, pensadas para copiar y pegar directamente durante una competencia. Cada archivo suele incluir la lógica del algoritmo lista para adaptarse al problema específico.

## Sobre CodeDeGrace

CodeDeGrace es un equipo de programación competitiva de la Universidad Tecnológica de Pereira, que participa en el ICPC.
